# 마크다운 지우개

서식은 지우고, 내용만 깨끗하게.

마크다운을 붙여넣으면 제목, 강조, 링크, 표 같은 문법 기호를 지우고 읽기 편한 일반 텍스트로 바꾸는 정적 웹 도구입니다.

## 특징

- 입력과 동시에 일반 텍스트로 변환
- 제목, 인용, 링크, 이미지, 표, 목록, 체크 항목, 코드 지원
- 복사·붙여넣기·예시·전체 지우기 제공
- 서버 전송, 계정, 추적 스크립트, AI 기능 없음
- 빌드 과정과 외부 라이브러리 없음

## 로컬 실행

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 배포

루트 디렉터리를 그대로 정적 호스팅하면 됩니다. GitHub Pages, Vercel, Netlify에서 별도 빌드 명령 없이 배포할 수 있습니다.
